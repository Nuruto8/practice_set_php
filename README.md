Requirements

XAMPP Control Panel v3.3.0 (Used to run Apache and MySQL locally)

PHP 8.0 or higher

Visual Studio Code (Recommended IDE)

How to Run the PHP Files

Install and launch XAMPP Control Panel v3.3.0.

Start the Apache server.
